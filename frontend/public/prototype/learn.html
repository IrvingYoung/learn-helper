数组 (Array)

入门
📖

概念讲解

数组是一种线性数据结构，它用一组连续的内存空间来存储一组具有相同类型的数据。

数组支持通过下标随机访问元素，时间复杂度为 O(1)。但插入和删除操作需要移动元素，时间复杂度为 O(n)。

核心特性

  • 连续的内存空间
  • 相同的数据类型
  • 支持随机访问（O(1)）
  • 插入/删除效率低（O(n)）
🔑

关键要点

随机访问 O(1) 插入 O(n) 删除 O(n) 动态扩容 内存连续性
💻

代码示例

Python
# 数组的基本操作
arr = [1, 2, 3, 4, 5]

# 随机访问
print(arr[2])  # O(1)

# 插入元素
arr.insert(2, 10)  # O(n)

# 删除元素
arr.pop(3)  # O(n)

# 遍历
for i in arr:
    print(i)
