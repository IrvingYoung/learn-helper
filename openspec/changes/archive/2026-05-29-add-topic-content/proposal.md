# Proposal: 添加知识点详细讲解内容

## Why

当前系统的知识点和练习题仅有标题和简介，缺乏完整的教学资料。用户学习时无法获得详细的讲解、代码示例和常见错误提示，严重影响学习效果。需要为每个知识点和练习题补充完整的讲解内容，形成可独立学习的教学资料库。

## What Changes

### 数据库改动

- **topics 表扩展**：
  - `content`: 详细讲解（Markdown 格式）
  - `code_examples`: 代码示例（JSON 格式，支持 Python/Go 等多语言）
  - `common_mistakes`: 常见错误（JSON 数组）

- **exercises 表扩展**：
  - `solution_detail`: 详细解法（含多种解法）
  - `common_errors`: 常见错误（JSON 数组）

### 数据生成

- 使用 AI 生成所有现有知识点的详细讲解内容
- 每道练习题附带完整解法和常见错误提示
- 代码示例以 Python 为主（算法教学通用语言）

### 前端改动

- 知识点详情页展示完整讲解内容
- 练习题详情页展示详细解法和代码
- Markdown 渲染支持（代码高亮、表格、列表）

## Capabilities

### New Capabilities

- `topic-content`: 知识点详细讲解内容管理
  - 包含概念定义、核心操作、复杂度分析、常见变形、应用场景、常见错误
  - 支持多语言代码示例
  - 关联练习题推荐

- `exercise-detail`: 练习题详细解法内容管理
  - 包含多种解法分析、完整代码、时间/空间复杂度推导
  - 分层提示（3层，从思路到实现）
  - 常见错误和陷阱标注

## Impact

- **数据库**: 新增 5 个字段，需数据迁移
- **后端**: topics/exercises 查询返回新增字段
- **前端**: 知识点和练习详情页需要渲染 Markdown 和代码高亮
- **内容**: 13 个知识点 + 6 道练习题需要生成讲解数据
